# Whack-A-Prof 🎓

A browser-based whack-a-mole style game built as a group project. Players click on professors as they pop up to score points before the timer runs out.

## My Contributions
I was responsible for:
- **Score tracking** — logic for incrementing and displaying the score when a mole is successfully whacked
- **Timer system** — starting, stopping, and resetting the countdown timer
- **Game state management** — handling the start, pause, resume, and end game flow

## How to Play
1. Click **Start** to begin the game
2. Click on a professor as they appear to score a point
3. Try to get as many points as possible before the 30 second timer runs out

## How to Run
1. Make sure you have [Node.js](https://nodejs.org) installed
2. Run `npm install` in the project folder
3. Run `node server.js`
4. Open your browser and go to `http://localhost:3000`
